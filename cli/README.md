# CLI

## Commands

```
$dualbox enc --file0 <file_path_0> --file1 <file_path_1> --key0 <key_0> --key1 <key_1>
$dualbox enc --file0 <file_path_0> --file1 <file_path_1> --key-file <key_file_path>
$dualbox dec --file <file_path> --key <key>
```
